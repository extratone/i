# MusicHarbor Artist List Shortcut Documentation 
## Returns a text list of all Artists in one’s current MusicHarbor library in Quick Look.
* [**RoutineHub Page**](https://routinehub.co/shortcut/8935/)
* [iCloud Share Link](https://www.icloud.com/shortcuts/676b15a473db457ebbc983da67f6d892)
***
## About
A very minor modification to the “Get Artists” Shortcut included with the MusicHarbor app which returns a text list of all Artists in one’s MusicHarbor library via Quick Look instead of a Preview notification.
